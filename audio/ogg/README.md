# Finale OGG-Audios

Hier kommen die finalen TipTalk-Audios hinein.

Format:

- Ogg Vorbis
- mono
- 22050 Hz
- Dateiname exakt gemaess `../../data/oid_mapping.csv`

Beispiel:

```text
china_mandarin_01_hallo_wie_geht_es_dir.ogg
```

Solange die finalen Dateien fehlen, nutzt `tiptalk.yaml` Entwicklungsansagen im `speak`-Abschnitt, sofern die lokale tttool-Version Text-to-Speech erzeugen kann.
